# 🍔 Food App (Firebase Powered)

تطبيق طعام متكامل مبني باستخدام Firebase، يتيح للمستخدمين تصفح الأطعمة حسب التصنيفات، إضافة المنتجات إلى السلة، وإتمام الطلب بسهولة، مع لوحة تحكم خاصة بالأدمن لإدارة المنتجات.

---

## 🚀 المميزات

- 📱 واجهة مستخدم بسيطة وسلسة
- 🍕 عرض المنتجات حسب التصنيفات (Categories)
- 🔍 تصفح الأطعمة مع تفاصيل كل منتج
- 🛒 سلة مشتريات (Cart) لإضافة وحذف وتعديل الكميات
- 🔥 ربط كامل مع Firebase (Realtime / Firestore)
- 🖼️ رفع وتخزين الصور باستخدام Firebase Storage
- 👨‍💻 لوحة تحكم للأدمن لإدارة المنتجات
- ⚡ تحديث البيانات بشكل فوري (Real-time updates)

---

## 🧩 التقنيات المستخدمة

- Flutter
- Firebase Authentication 
- Cloud Firestore
- Firebase Storage
- State Management ( Bloc )

---

## 🍱 مميزات التطبيق

### 🍕 التصنيفات (Categories)
يعرض التطبيق الأطعمة ضمن تصنيفات مثل:
- برغر 🍔
- بيتزا 🍕
- مشروبات 🥤
- حلويات 🍰

---

### 🥗 المنتجات
كل منتج يحتوي على:
- الاسم
- السعر
- الصورة
- وصف مختصر

---

### 🛒 السلة (Cart)
- إضافة منتجات إلى السلة
- تعديل الكمية
- حذف منتجات
- حساب السعر الإجمالي تلقائياً

---

## 🛠️ لوحة التحكم (Admin Panel)

تم إنشاء صفحات خاصة بالأدمن وربطها بنفس قاعدة بيانات Firebase المستخدمة في التطبيق، مما يسمح بإدارة المحتوى بشكل كامل.

### 👨‍💻 صلاحيات الأدمن:
- ➕ إضافة منتجات جديدة
- ✏️ تعديل المنتجات
- 🗑️ حذف المنتجات
- 📂 إدارة التصنيفات

### 🔗 الربط مع Firebase:
- Cloud Firestore لإدارة البيانات بشكل مباشر
- Firebase Storage لرفع وتحديث الصور

### ⚡ مميزات لوحة الأدمن:
- تحديث فوري للبيانات داخل التطبيق
- واجهة بسيطة وسهلة الاستخدام
- نفس قاعدة البيانات بين المستخدم والأدمن لضمان التزامن
