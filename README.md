# 🍔 Food App (Firebase Powered)

تطبيق طعام متكامل مبني باستخدام Firebase، يتيح للمستخدمين تصفح الأطعمة حسب التصنيفات، إضافة المنتجات إلى السلة، وإتمام الطلب بسهولة، مع لوحة تحكم خاصة بالأدمن لإدارة المنتجات.

---

## 🚀 المميزات

- 📱 واجهة مستخدم بسيطة وسلسة
- 🍕 عرض المنتجات حسب التصنيفات (Categories)
- 🔍 تصفح الأطعمة مع تفاصيل كل منتج
- 🛒 سلة مشتريات (Cart) لإضافة وحذف وتعديل الكميات
- 🔥 ربط كامل مع Firebase (Realtime / Firestore)
- 🖼️ رفع وتخزين الصور باستخدام Firebase Storage
- 👨‍💻 لوحة تحكم للأدمن لإدارة المنتجات
- ⚡ تحديث البيانات بشكل فوري (Real-time updates)

---

## 🧩 التقنيات المستخدمة

- Flutter
- Firebase Authentication 
- Cloud Firestore
- Firebase Storage
- State Management ( Bloc )
